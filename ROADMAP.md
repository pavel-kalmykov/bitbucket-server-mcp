# Roadmap

There is no detailed 12-month plan. We already cover a fair chunk of the
typical Bitbucket operations (PRs, branches, tags, commits, search, code
review, insights, files) and the project keeps growing organically as new
needs come up, either from the author or from the community.

Once the project hits 90 days we will start stabilising toward 1.0.0 and
getting out of pre-release.

## Non-goals

- **Bitbucket Cloud (bitbucket.org)**: Different API. This is Server/Data Center only.
- **Built-in LLM**: This is an MCP server, not an agent.
- **Jira / Confluence**: They go in separate MCPs.
- **GUI / dashboard**: Headless, STDIO or HTTP transport only.
